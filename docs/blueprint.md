# **App Name**: CareerPilot AI

## Core Features:

- AI-Powered Job Analysis: Parse job descriptions from URLs or pasted text, calculate an AI match score against the user's profile and resume, identify matched and missing skills, and generate actionable improvement suggestions for better fit.
- Resume & Profile Management: Allow users to upload, parse, and store PDF/DOCX resumes. Maintain a structured user profile with professional details, key skills, experience, and preferred roles, which is utilized for all AI analyses.
- Tailored Application Generator: Generate job-specific resume improvement suggestions (ATS keywords, bullet point rewrites, summary optimization) and personalized cover letters with tone selection, offering editable output for user customization.
- Comprehensive Application Tracker: Provide a central dashboard to save jobs from various sources, track application statuses (saved, applied, interview, offer), add personal notes, and set reminder dates for follow-ups and interviews.
- Browser Extension for Seamless Import: A lightweight browser extension that detects job details on web pages, extracts content for quick AI analysis, and allows users to save jobs directly to their tracker or open full analysis in the web app.
- AI-Driven Interview Preparation: Generate likely interview questions, categorized by type (behavioral, technical, role-fit), directly from the job description, and provide AI-generated guidance bullets for constructing effective answers.
- Interactive AI Career Assistant: A chat-style AI tool within the platform that offers context-aware advice on resume optimization, job search strategies, skill development, and interview tips, leveraging the user's profile and saved jobs.

## Style Guidelines:

- The primary color is a deep, trustworthy blue (#296699), chosen to convey professionalism and clarity, providing strong contrast against lighter elements.
- The background color is a very light, desaturated cool grey (#EBF1F5), promoting a clean, calm, and focus-oriented interface for reducing job seeker stress.
- An accent color of vibrant violet (#702AE6) is used for call-to-action elements and highlights, offering a distinct and energetic contrast to the primary blue while maintaining a professional aesthetic.
- Headlines use 'Space Grotesk' (proportional sans-serif) for a modern, tech-assisted, and impactful visual appeal suitable for an AI-focused product.
- Body text is set in 'Inter' (grotesque-style sans-serif) for its neutral, objective, and highly legible characteristics, ensuring clarity in dense blocks of job descriptions and AI-generated content.
- Code snippets, if presented, will utilize 'Source Code Pro' (monospaced sans-serif) for consistent and readable display of technical examples or skill recommendations.
- A dashboard-centric layout emphasizes organized data, with clear visual hierarchy to quickly convey application status, reminders, and key AI insights.
- Utilize clean, outline-style icons that maintain a modern and minimalistic aesthetic, ensuring universal readability across web, extension, and mobile platforms.
- Subtle, purpose-driven animations should be employed for state changes and content loading, enhancing the user experience without causing distraction or delays.

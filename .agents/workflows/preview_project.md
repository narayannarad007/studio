# Preview Project

Start the Next.js development server to test your changes.

1. **Verify Environment**
   Ensure Node.js (>= 20) is installed.

2. **Start Dev Server**
   Start the development server with Turbopack.
   ```bash
   npm run dev
   ```

3. **Open Preview**
   Navigate to `http://localhost:3000` in your browser.

4. **Hot Reloading**
   The server will automatically update as you make changes to files in the `src` directory.
